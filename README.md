# Azure & Wintel Operations Lab

A comprehensive repository documenting hands-on enterprise cloud operations, Windows Server administration, and system remediation. Every lab here represents actual deployment, configuration, and troubleshooting managed through the Azure Portal, CLI, and PowerShell.

## 🚀 Lab Progress & Domains

| Domain | Focus Area | Status |
| :--- | :--- | :--- |
| **01 - Identity & Governance** | Entra ID, RBAC Roles, Azure Policies | 🔄 In Progress |
| **02 - Compute & Wintel** | VM Provisioning, Active Directory, Server Remediation | ⏳ Coming Soon |
| **03 - Linux Server Security** | OS Installation, SSH Key Auth, System Hardening | ⏳ Coming Soon |
| **04 - Hybrid Networking** | Virtual Networks, Network Security Groups (NSGs) | ⏳ Coming Soon |
| **05 - Monitoring & Observability** | Azure Monitor, Dynatrace, Log Analytics | ⏳ Coming Soon |

## 🛠️ Infrastructure Methodology

*   **Production Mindset:** Building architecture with strict adherence to the principle of least privilege.
*   **Troubleshooting Focus:** Every lab documents not just the successful configuration, but the error logs, misconfigurations, and specific resolution steps encountered along the way.
*   **Hybrid Integration:** Bridging the gap between Wintel server administration and public cloud infrastructure.
